SOC design, configuration and operations project

This project is about designing, configuring and commissioning a typical Cybersecurity Operations Center (SOC). Also use it monitor the network and all the devices in the system, respond to unusual activities and respond accordingly - Triage. The SIEM tool used in this solution Design is WAZUH server and its agents on the endpoints. A brute-force attack was simulated, detected, incident response activated and fully investigated. Full report was delivered with great insights and actionable recommendations.

Malware Analysis Project

This project is on static and dynamic Malware analysis using Ghidra, IDA pro and virustotal. Also monitoring malware detection on the SIEM tool dashboard and responding appropriately. The Static analysis refers to application reverse software engineering(RSE) methodoly, dynamy analysis refers to the used of web-based tools/platforms to analyze malicious files and URLs. The WAZUH as an open-source SIEM tool has all these capabilities to to catch malwares in realtime and display them in the dashboard logs with alerts/notifications for prompt response.
